# Public projects and contributions

This is a curated map, not a claim that every repository under `forge-builder` is maintained by Aurel.

## Status vocabulary

| Status | Meaning |
|---|---|
| **Active** | Owned, currently maintained, and backed by a stated verifier |
| **Staged** | Real draft or branch exists; not live on the destination default branch |
| **Candidate** | Public source exists but needs health, license, claim, or maintenance review |
| **Source shelf** | Fork or external source retained for study/contribution; not adopted capability |
| **Paused** | Deliberately not receiving current production effort |
| **Merged contribution** | Accepted upstream work with a public pull-request record |
| **Archived** | Intentionally frozen and excluded from current routing |

## Source shelves

Repositories prefixed `aurel-` that are forks remain source shelves unless this registry explicitly promotes them. A fork name records routing and provenance; it does not prove adoption, maintenance, or endorsement.

Private Studio, customer, runtime, and recovery bodies are intentionally absent from this public map.
